# En-tête et pied de page (header & footer)

Ce document décrit les patterns de templates header et footer utilisés sur l'ensemble du portail Audiodescription. 
Ces templates Twig implémentent le Système de Design de l'État français (DSFR) et fournissent une navigation, un branding et un contenu de pied de page cohérents sur toutes les pages.

Pour les informations sur les fils d'Ariane, voir [Fil d'ariane](5.6.1-Breadcrumb-Builders.md). 
Pour le preprocessing du thème qui prépare les données pour ces templates, voir [Preprocessing du thème](5.2.2-Theme-Preprocessing.md).

## Vue d'ensemble

L'en-tête et le pied de page sont implémentés comme des templates Twig réutilisables, invoqués depuis le template de page principal `ad_theme/templates/system/page.html.twig`. 
Les 2 templates acceptent des données structurées via des variables de template et restituent du HTML conforme au DSFR avec des attributs d'accessibilité appropriés et un comportement responsive.

**Composants clés :**

- **Pattern header** (`pattern-header.html.twig`) : logo, titre du service, menu de navigation, modal de recherche et liens outils
- **Pattern footer** (`pattern-footer.html.twig`) : navigation du footer, branding, logos partenaires et informations légales
- **Template de page** (`page.html.twig`) : orchestre header, footer et régions de contenu principal

## Intégration dans le template de page

### Liens d'évitement (skiplinks)

Le template de page inclut des liens d'évitement accessibles avant le header :

| Texte du lien | Cible | Rôle |
|---|---|---|
| "Content" | `#contenu` | Accès au contenu principal |
| "Recherche" | `#search` | Accès à la recherche |
| "Menu" | `#navigation-main` | Accès au menu de navigation |
| "Footer" | `#footer` | Accès au pied de page |

### Flux des régions

La zone de contenu principal utilise les propriétés CSS `order` pour contrôler le positionnement des sidebars :

- Contenu : `order: 2` (milieu)
- Sidebar gauche : `order: 1` (gauche)
- Sidebar droite : `order: 3` (droite)
